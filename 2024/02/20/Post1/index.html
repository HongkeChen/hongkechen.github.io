准备

Win10/Win11电脑一台（请确保磁盘有足够空间 建议留出10g以上的剩余空间），数据线一根，iEnglish4平板（平板后盖附着进网许可印有“9011”字样）一台

环境配置

Mtk Universal 解锁工具

下载mtk universal v5工具，解bl用，或者你自己有别的解bl工具也行。解压后进入文件夹打开krypton开头的exe，然后点driver install x64安装驱动

Mtkclient

下载mtkclient-1.52 并解压缩

Python

去Python官网下载合适的Python安装包并安装（必须要3.9或以上），记得在安装第一步勾选add to path，或者直接去微软商店里安装Python

驱动

下载mtk驱动解压缩并安装，记得一路点允许安装此驱动程序

建议配置完环境后重启一次电脑

基础教程

解bl

先进入mtk universal v5那个文件夹，打开里面krypton开头的exe，点bootloader unlock，然后点start。这个时候确保平板已经完全关机，然后同时按住音量加减并迅速插线，0.5秒后可以松开按键
如果不出意外的话此时bl已经解开。如果工具报错的话就拔下平板并连接充电器，等屏幕上出现电池图标以后拔线。此时平板已经回到完全关机状态，
记得关掉工具重新打开然后再试一遍。

提取system和boot

打开mtkclient-1.52文件夹，在文件管理器窗口空白处右键，然后点右键菜单里的在此处打开命令窗口，（关于Python：第一次运行需输入

并回车然后等待环境安装完成微软商店下载Python的无需执行此步）（关于工具：第一次运行需输入pip3 install -r requirements.txt并回车等待环境安装完成，再进行下一步操作）输入python mtk r system,boot system.img,boot.img
并回车，输完指令后像第一步一样按音量上下键同时插线连接平板，等待进度条走完即可成功提取boot和system分区（如果工具报错请尝试重新打开命令窗口或者重启电脑，记得先像第一步里充电让平板回到完全关机）。提取出来的文件在mtkclient-1.52这个文件夹里。

未命名段落

下载新系统文件链接：

https://www.123pan.com/s/hXOrVv-v2SK3 提取码：H5TC4
https://www.123pan.com/s/hXOrVv-KnbK3.html 提取码:iEng

刷入新的system.img下载完的系统文件改名
system.img将system.img移动到mtkclient-1.52文件夹开始刷入：打开mtkclient-1.52文件夹，在窗口空白处右键，点右键菜单里的在此处打开命令窗口，然后输入python mtk w system system.img然后回车 刷入system分区，输完指令后像第一步一样按音量键同时插线连接平板，等待进度条走完即可（如果工具报错请尝试重新打开命令窗口或者重启电脑，记得先像第一步里充电让平板回到完全关机）。刷完了切记不要马上拔线开机，要先清空所有数据。
